# End-to-End DevOps EKS Blog App

This project demonstrates an end-to-end DevOps workflow using Docker, Docker Hub, and Kubernetes.

## Current Workflow

Flask App → Docker Image → Docker Hub → Kubernetes Deployment → NodePort Service

## Tools Used

- Python Flask
- Docker
- Docker Hub
- Kubernetes
- kubectl
- Docker Desktop
